Gradient

Gradient is a custom-designed font that I made purely because I love typefaces and creative design. The goal of the font was to be simple and geometric. Each glyph was created in Adobe Illustrator starting with shapes and then breaking them down into recognizable English characters. After all of the letters were created, spacing and kerning were done to ensure a uniform flow of the characters. This is a project I had been envisioning for years and began an early prototype back in 2017. Since then, I've learned much more about design workflows and project management. Gradient was designed to be fun and expressive, while still having a professional look to it. I think a custom font is a great way to stand out and brand yourself, and I am beyond excited to implement it in the many projects I've got coming in the future.

Linux Web Server

I enjoy challenging myself in new ways all the time. Recently, I decided I wanted to learn how to set up a custom Linux web server. I chose to use CentOS 8 in combination with Nginx. This was a learning experience for me, as I got to learn how to open and secure a port to the internet. After making the necessary preparations, I was able to get the whole thing online and even connected to a custom domain! I was thrilled by the concept and the networking required. However, while it was a great experiment, I elected to back-up and take the server down due to security concerns and maintenance requirements.
